# Nonprofit Impact Calculator

A free and open-source web application that helps nonprofit organizations measure operational performance, financial sustainability, and social impact.

---

## Features

- Cost per beneficiary calculation
- Revenue coverage ratio
- Financial sustainability ratio
- Program achievement rate
- Responsive interface
- Lightweight and fast
- Open source (MIT License)

---

## Roadmap

- Dashboard
- Interactive charts
- PDF export
- CSV export
- Arabic & English interface
- GitHub Pages deployment
- Additional nonprofit KPIs

---

## Technologies

- HTML5
- CSS3
- JavaScript

---

## Getting Started

1. Clone the repository.
2. Open `index.html` in your browser.
3. Start using the calculator.
